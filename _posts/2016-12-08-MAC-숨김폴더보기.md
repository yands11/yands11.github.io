# MAC - 숨김폴더 보기

---

## 숨김폴더 해제  

1. 터미널을 킨다  
2. 다음을 입력한다  

```  
defaults write com.apple.finder AppleShowAllFiles -bool true  
```

3. 다음을 입력한다 (기존 Finder 창이 꺼지고 다시 실행된다)  

```  
killall Finder  
```

## 원상복구  

1. 터미널을 킨다  
2. 다음을 입력한다  

```  
defaults write com.apple.finder AppleShowAllFiles -bool false  
```

3. 다음을 입력한다 (기존 Finder 창이 꺼지고 다시 실행된다)  

```  
killall Finder   
```
